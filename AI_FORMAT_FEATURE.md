# AI 检查排版功能

## 功能说明

在笔记编辑器中新增了 "AI 检查排版" 功能，可以自动优化你的 Markdown 文档。

## 使用方法

1. 在编辑器中输入或粘贴你的内容
2. 点击工具栏旁边的 **"AI 检查排版"** 按钮（带有 ✨ 图标）
3. AI 会自动：
   - 修正语法和拼写错误
   - 优化段落结构
   - 统一标题层级
   - 改善列表格式
   - 确保 Markdown 语法正确

## 视觉效果

- **处理中**：按钮周围会出现炫酷的彩虹霓虹灯边框效果
- **流式输出**：优化后的内容会实时显示在编辑器中
- **完成提示**：处理完成后会显示成功提示

## 技术实现

- 使用 DeepSeek AI 进行智能文本处理
- 支持流式响应，实时显示结果
- 彩虹霓虹灯动画使用 CSS keyframes 实现
- 完全集成到现有编辑器工作流中

## 示例

**原文：**
```
# 标题
这是一段没有格式的文本内容
- 列表项1
-列表项2
```

**优化后：**
```
# 标题

这是一段格式优化后的文本内容。

- 列表项 1
- 列表项 2
```

## 注意事项

- 需要配置 DeepSeek API Key
- 需要登录才能使用
- 处理时间取决于内容长度和 API 响应速度
