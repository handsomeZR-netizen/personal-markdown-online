## 🌟 最新功能亮点

### 1. 离线同步 🆕
- 离线编辑笔记，无需网络连接
- 联网后自动同步到云端
- 智能冲突解决
- IndexedDB 本地存储
- 同步状态实时显示

### 2. 数学公式支持 🆕
- 完整的 LaTeX 数学公式渲染
- 支持行内公式：`$E=mc^2$`
- 支持块级公式：`$$\sum_{i=1}^{n} x_i$$`
- 支持分数、根号、求和、积分、矩阵等

### 3. 图片拖拽上传 🆕
- 直接拖拽图片到编辑器
- 支持粘贴上传 (Ctrl/Cmd+V)
- 实时进度显示
- 批量上传支持
- 自动插入 Markdown 图片语法

### 4. 文件夹侧边栏 ✨
- 在笔记列表左侧显示文件夹树
- 支持展开/折叠
- 显示笔记数量
- 点击筛选笔记
- 快速创建文件夹

### 5. 加载动画系统 🆕
- Orbit 轨道动画
- Pulse 脉冲动画
- Dots 点阵动画
- Wave 波浪动画
- 自动应用于所有异步操作

### 6. 快捷键支持 ⚡
- Ctrl+S 保存笔记
- Ctrl+K 快速搜索
- Ctrl+B 加粗文本
- Ctrl+I 斜体文本
- Ctrl+/ 插入注释
- 更多快捷键...

## 🚀 快速开始指南

1. **创建笔记** - 点击"新建笔记"按钮，开始使用 Markdown 编辑器
2. **文件夹管理** - 在笔记列表左侧创建文件夹，拖放笔记进行组织
3. **图片上传** - 在编辑器中直接粘贴图片或拖放图片文件
4. **数学公式** - 使用 `$E=mc^2$` 输入行内公式，`$$公式$$` 输入块级公式
5. **导出笔记** - 打开任意笔记，点击编辑器工具栏的导出按钮
6. **版本历史** - 在编辑器工具栏点击历史按钮，查看和恢复历史版本
7. **公开分享** - 在笔记页面点击分享按钮，生成公开链接
8. **移动端** - 在设置中安装 PWA 应用，获得原生应用体验
