# 快速开始指南

## 🎉 欢迎使用笔记管理平台！

所有功能都已完全集成，你可以立即开始使用。

## 🚀 5 分钟快速上手

### 1️⃣ 创建第一篇笔记
```
Dashboard → 点击"新建笔记" → 开始写作
```

### 2️⃣ 探索所有功能
```
顶部导航 → 点击"功能" → 查看 11 个核心功能
```

### 3️⃣ 使用 AI 助手
```
顶部导航 → 点击"AI 功能" → 配置 API → 开始使用
```

### 4️⃣ 邀请协作
```
打开笔记 → 点击"分享" → 输入邮箱 → 开始协作
```

### 5️⃣ 移动端使用
```
设置 → PWA 设置 → 安装应用 → 离线使用
```

## 💡 使用技巧

1. **使用模板**: 访问 `/templates` 快速创建结构化笔记
2. **AI 标签**: 让 AI 自动为笔记添加标签
3. **离线编辑**: 启用离线模式，随时随地访问

## 🔧 首次配置

### AI 功能（可选）
1. 访问 `/settings`
2. 展开"AI API 配置"
3. 选择提供商并输入 API 密钥

### 移动端（推荐）
1. 访问 `/settings/pwa`
2. 点击"安装应用"
3. 添加到主屏幕

### Webhook（可选）
1. 访问 `/settings/webhooks`
2. 输入 Webhook URL
3. 选择要监听的事件
