# 无障碍访问检查清单

本文档提供了一个全面的无障碍访问检查清单，帮助确保应用符合 WCAG 2.1 AA 标准。

## 键盘导航

### 基本要求

- [ ] 所有交互元素可通过键盘访问
- [ ] Tab 键顺序符合逻辑
- [ ] 焦点指示器清晰可见
- [ ] 支持 Shift+Tab 反向导航
- [ ] 没有键盘陷阱（用户可以离开任何元素）

### 快捷键

- [ ] Enter 键激活按钮和链接
- [ ] Space 键激活按钮
- [ ] Escape 键关闭对话框和弹出层
- [ ] 箭头键导航列表和菜单
- [ ] Home/End 键跳转到开始/结束

### 焦点管理

- [ ] 打开对话框时焦点移到对话框内
- [ ] 关闭对话框时焦点返回触发元素
- [ ] 模态对话框内焦点被困住
- [ ] 跳过链接可跳转到主内容
- [ ] 焦点不会意外丢失

## 屏幕阅读器支持

### 语义化 HTML

- [ ] 使用正确的标题层级（h1-h6）
- [ ] 使用语义化元素（nav, main, article, section）
- [ ] 列表使用 ul/ol/li
- [ ] 表单使用 form, label, input
- [ ] 按钮使用 button 元素

### ARIA 属性

- [ ] 所有交互元素有可访问名称
- [ ] 图标按钮有 aria-label
- [ ] 表单输入有关联的 label
- [ ] 必填字段标记 aria-required
- [ ] 错误字段标记 aria-invalid
- [ ] 动态内容使用 aria-live
- [ ] 对话框使用 role="dialog" 和 aria-modal
- [ ] 树形结构使用 role="tree" 和 role="treeitem"
- [ ] 展开/折叠元素使用 aria-expanded
- [ ] 当前页面使用 aria-current="page"

### 可访问名称

- [ ] 所有图片有 alt 文本
- [ ] 装饰性图片使用 alt="" 或 aria-hidden="true"
- [ ] 链接文本描述目标
- [ ] 按钮文本描述操作
- [ ] 表单标签清晰描述输入

### 状态和属性

- [ ] 加载状态使用 aria-busy
- [ ] 禁用元素使用 aria-disabled
- [ ] 选中状态使用 aria-checked
- [ ] 展开状态使用 aria-expanded
- [ ] 隐藏内容使用 aria-hidden

## 颜色和对比度

### 对比度要求

- [ ] 正常文本对比度 ≥ 4.5:1
- [ ] 大文本（18pt+）对比度 ≥ 3:1
- [ ] UI 组件对比度 ≥ 3:1
- [ ] 焦点指示器对比度 ≥ 3:1

### 颜色使用

- [ ] 不仅依赖颜色传达信息
- [ ] 错误/成功消息有图标或文本
- [ ] 链接有下划线或其他视觉区分
- [ ] 图表有模式或标签

### 主题支持

- [ ] 支持深色模式
- [ ] 支持高对比度模式
- [ ] 颜色可自定义

## 文本和内容

### 可读性

- [ ] 字体大小至少 16px
- [ ] 行高至少 1.5
- [ ] 段落间距适当
- [ ] 文本可缩放到 200%
- [ ] 文本不会被截断

### 语言

- [ ] HTML lang 属性正确
- [ ] 多语言内容标记 lang
- [ ] 术语有解释
- [ ] 缩写有全称

### 内容结构

- [ ] 标题层级正确
- [ ] 列表结构清晰
- [ ] 表格有表头
- [ ] 引用有引用标记

## 表单

### 标签和说明

- [ ] 所有输入有关联标签
- [ ] 必填字段明确标记
- [ ] 占位符不替代标签
- [ ] 帮助文本关联到输入
- [ ] 错误消息关联到输入

### 验证和错误

- [ ] 错误消息清晰具体
- [ ] 错误位置明确
- [ ] 提供修正建议
- [ ] 错误汇总在表单顶部
- [ ] 成功消息可访问

### 输入类型

- [ ] 使用正确的 input type
- [ ] 自动完成属性正确
- [ ] 日期选择器可键盘操作
- [ ] 下拉菜单可搜索

## 移动端

### 触摸目标

- [ ] 触摸目标至少 44x44 像素
- [ ] 触摸目标间距至少 8 像素
- [ ] 重要操作易于触及
- [ ] 避免误触

### 手势

- [ ] 提供替代操作方式
- [ ] 手势有说明
- [ ] 支持单指操作
- [ ] 避免复杂手势

### 缩放

- [ ] 允许捏合缩放
- [ ] 不设置 maximum-scale=1
- [ ] 不设置 user-scalable=no
- [ ] 文本可缩放

### 方向

- [ ] 支持横屏和竖屏
- [ ] 方向改变不丢失内容
- [ ] 不强制特定方向

## 多媒体

### 图片

- [ ] 所有图片有 alt 文本
- [ ] 复杂图片有详细描述
- [ ] 装饰图片标记为装饰性
- [ ] 图片加载失败有提示

### 视频

- [ ] 提供字幕
- [ ] 提供音频描述
- [ ] 播放器可键盘控制
- [ ] 不自动播放

### 音频

- [ ] 提供文字稿
- [ ] 播放器可键盘控制
- [ ] 音量可调节
- [ ] 不自动播放

## 动画和过渡

### 动画控制

- [ ] 支持 prefers-reduced-motion
- [ ] 动画可暂停
- [ ] 动画不超过 5 秒
- [ ] 闪烁频率 < 3 次/秒

### 过渡

- [ ] 过渡时间适中
- [ ] 过渡不影响可用性
- [ ] 加载动画有说明

## 错误处理

### 错误消息

- [ ] 错误消息清晰
- [ ] 错误位置明确
- [ ] 提供解决方案
- [ ] 使用 role="alert"

### 错误预防

- [ ] 重要操作需确认
- [ ] 提供撤销功能
- [ ] 自动保存草稿
- [ ] 表单数据不丢失

## 时间限制

### 超时

- [ ] 超时前有警告
- [ ] 可延长时间
- [ ] 可关闭超时
- [ ] 保存用户数据

### 自动刷新

- [ ] 不自动刷新
- [ ] 刷新前有提示
- [ ] 可暂停刷新
