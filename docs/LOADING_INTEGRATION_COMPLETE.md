## 🎯 关键特性

### 1. 防止重复提交
所有按钮在加载时自动禁用，防止用户重复点击。

### 2. 全局加载覆盖层
重要操作（保存、删除）显示全屏加载，阻止用户进行其他操作。

### 3. 内联加载
按钮内显示加载动画，不影响页面布局。

### 4. 页面级加载
使用 Next.js 的 `loading.tsx` 文件，自动处理页面切换时的加载状态。

### 5. 骨架屏
列表页面使用骨架屏，提供更好的加载体验。

## 💡 扩展建议

如果你想在其他地方添加加载动画：

### 快速添加方法

**1. 按钮加载**
```tsx
import { LoadingButton } from '@/components/ui/loading-button';

<LoadingButton 
  loading={isLoading}
  loaderVariant="orbit"
  onClick={handleClick}
>
  保存
</LoadingButton>
```

**2. 全局加载**
```tsx
import { useLoading } from '@/hooks/use-loading';

const { showLoading, hideLoading } = useLoading();

showLoading('处理中...', 'orbit');
await doSomething();
hideLoading();
```

**3. 容器加载**
```tsx
import { LoadingContainer } from '@/components/ui/with-loading';

<LoadingContainer isLoading={loading} variant="wave">
  <Content />
</LoadingContainer>
```
