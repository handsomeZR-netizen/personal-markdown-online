# 笔记管理平台 - 用户指南

欢迎使用我们的笔记管理平台！这是一个功能强大的协作知识库系统。

## 🚀 快速开始

### 首次使用
1. 注册账号并登录
2. 访问 Dashboard 查看概览
3. 点击"新建笔记"创建第一篇笔记
4. 探索"功能导航"了解所有可用功能

### 界面导航
- **顶部导航栏**: 快速访问主要功能
- **搜索栏**: 搜索笔记和文件夹
- **功能按钮**: 访问 `/features` 查看所有功能
- **移动端**: 底部导航栏提供快捷访问

## 📝 核心功能

### 1. 笔记编辑

#### 创建笔记
- 点击"新建笔记"按钮
- 或使用快捷键 `Ctrl/Cmd + N`
- 或从模板创建

#### Markdown 编辑
- 支持完整的 Markdown 语法
- 实时预览（桌面端并排显示）
- 移动端可切换编辑/预览模式

#### 编辑器工具栏
```
[B] 加粗        Ctrl/Cmd + B
[I] 斜体        Ctrl/Cmd + I
[H] 标题        
[L] 链接        Ctrl/Cmd + L
[>] 引用
[•] 列表
[1] 有序列表
[`] 代码
```

#### 自动保存
- 编辑时自动保存草稿
- 离线时保存到本地
- 网络恢复后自动同步

### 2. 文件夹管理

#### 创建文件夹
1. 在 Dashboard 左侧点击"新建文件夹"
2. 输入文件夹名称
3. 可以创建嵌套文件夹

#### 组织笔记
- 拖放笔记到文件夹
- 拖放文件夹进行嵌套
- 右键菜单快速操作

### 3. 标签和分类

#### 添加标签
1. 在编辑器中找到"标签"区域
2. 选择现有标签或创建新标签
3. 支持多个标签

#### AI 标签建议
- 编辑笔记时自动分析内容
- 点击"AI 标签建议"获取推荐
- 一键添加建议的标签

#### 分类管理
- 为笔记选择分类
- 在 Dashboard 按分类筛选

### 4. 搜索功能

#### 基础搜索
- 使用顶部搜索栏
- 搜索笔记标题和内容
- 同时搜索文件夹

#### 高级搜索
- 访问 `/search` 页面
- 按标签筛选
- 按日期范围筛选
- 按分类筛选

#### AI 语义搜索
- 访问 `/ai` 页面
- 使用自然语言描述
- AI 理解语义并返回相关笔记

### 5. 协作功能

#### 邀请协作者
1. 打开笔记编辑页面
2. 点击"分享"按钮
3. 输入协作者邮箱
4. 选择权限级别：
   - **查看者**: 只能查看
   - **编辑者**: 可以编辑
   - **所有者**: 完全控制

#### 实时协作
- 查看其他人的光标位置
- 看到实时编辑内容
- 在线状态指示器
- 自动冲突解决

#### 公开分享
1. 在分享对话框中
2. 启用"公开访问"
3. 复制公开链接
4. 任何人都可以通过链接查看

### 6. 版本历史

#### 查看历史
1. 打开笔记编辑页面
2. 点击"版本历史"按钮
3. 浏览所有历史版本
4. 查看每个版本的详细内容

#### 恢复版本
1. 在版本历史中选择版本
2. 点击"恢复到此版本"
3. 确认恢复操作
4. 当前内容会被替换

### 7. 导出功能

#### 导出单个笔记
1. 打开笔记编辑页面
2. 点击"导出"按钮
3. 选择格式：
   - **Markdown**: 原始格式
   - **PDF**: 打印友好
   - **HTML**: 网页格式
4. 下载文件

#### 批量导出
- 在笔记列表选择多个笔记
- 点击批量操作
- 选择导出格式

### 8. 模板系统

#### 使用模板
1. 访问 `/templates`
2. 浏览可用模板：
   - 会议记录
   - 项目计划
   - 日报周报
   - 学习笔记
   - 等等...
3. 点击"使用模板"
4. 自动创建结构化笔记

#### 创建模板
1. 创建一个笔记
2. 设计好结构和格式
3. 保存为模板
4. 可以重复使用

### 9. AI 功能

#### AI 摘要
- 打开笔记详情页
- 自动生成 AI 摘要
- 点击"重新生成"更新摘要

#### AI 格式化
1. 在编辑器中选择文本
2. 点击"AI 格式化"按钮
3. AI 自动优化格式和结构

#### AI 问答
1. 访问 `/ai` 页面
2. 切换到"自然语言查询"
3. 用自然语言提问
4. AI 基于笔记内容回答

### 10. 移动端功能

#### PWA 安装
1. 访问 `/settings/pwa`
2. 点击"安装应用"
3. 添加到主屏幕
4. 像原生应用一样使用

#### 手势操作
- **左滑**: 删除笔记
- **右滑**: 标记为收藏
- **下拉**: 刷新列表
- **长按**: 显示快捷菜单

#### 底部导航
- **首页**: Dashboard
- **笔记**: 笔记列表
- **新建**: 快速创建
- **搜索**: 搜索功能
- **设置**: 应用设置

#### 离线模式
- 自动缓存笔记
- 离线时可以编辑
- 网络恢复后自动同步
- 冲突自动解决

### 11. Webhook 集成

#### 配置 Webhook
1. 访问 `/settings/webhooks`
2. 输入 Webhook URL
3. 选择要监听的事件：
   - 笔记创建
   - 笔记更新
   - 笔记删除
   - 协作者变更
4. 保存配置

#### 事件通知
- 自动发送 POST 请求
- 包含事件详情
- 可用于自动化工作流

## ⚙️ 设置和配置

### AI 配置
1. 访问 `/settings`
2. 展开"AI API 配置"
3. 选择 AI 提供商：
   - OpenAI
   - Anthropic
   - Google Gemini
   - 本地模型
4. 输入 API 密钥
5. 选择模型

### 离线设置
1. 访问 `/settings`
2. 展开"离线功能设置"
3. 配置：
   - 自动同步间隔
   - 草稿保存间隔
   - 冲突解决策略
   - 离线存储限制

### 缓存管理
1. 访问 `/settings`
2. 展开"缓存管理"
3. 查看缓存使用情况
4. 清理缓存数据

### 存储管理
1. 访问 `/settings/storage`
2. 查看存储配额
3. 查看大文件列表
4. 清理不需要的文件

## 🔧 故障排除

### 同步问题
- 检查网络连接
- 查看同步状态指示器
- 访问 `/settings/offline` 检查设置
- 手动触发同步

### 性能问题
- 清理缓存
- 减少离线存储的笔记数量
- 优化图片大小
- 使用搜索而不是浏览

### AI 功能不可用
- 检查 API 配置
- 验证 API 密钥
- 查看 API 状态指示器
- 尝试其他 AI 提供商

### 移动端问题
- 更新到最新版本
- 清除浏览器缓存
- 重新安装 PWA
- 检查存储权限
