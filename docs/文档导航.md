- ⚠️ 检查可选变量

---

#### 10. [vercel-env-setup.sh](./vercel-env-setup.sh) / [vercel-env-setup.ps1](./vercel-env-setup.ps1)
**用途**：自动配置 Vercel 环境变量
```bash
# Mac/Linux
npm run vercel:env

# Windows
npm run vercel:env:win
```
- 📋 从 .env.local 读取
- ⚙️ 自动配置到 Vercel
- ✅ 应用到所有环境
